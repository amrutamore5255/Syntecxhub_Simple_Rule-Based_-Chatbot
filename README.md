#  Simple Rule-Based Chatbot

##  Project Overview

The Simple Rule-Based Chatbot is a Python-based conversational application that responds to user inputs using predefined rules and pattern matching techniques. The chatbot can handle greetings, help requests, small talk, and answer basic questions from its built-in knowledge base.

This project demonstrates the fundamentals of Artificial Intelligence through a rule-based approach without using machine learning models.

---

##  Features

* Interactive command-line chatbot
* Handles greetings and small talk
* Provides help menu
* Answers AI-related questions using a knowledge base
* Logs conversation history with timestamps
* User-friendly and easy to extend
* Built entirely using Python

2. Navigate to the project folder

```bash

```

## Sample Interaction

```text
==================================================
      SIMPLE RULE-BASED CHATBOT
==================================================
Type 'help' for options.
Type 'bye' to exit.

You: hello
Bot: Hello! How can I help you today?

You: what is python
Bot: Python is a high-level programming language used in AI, web development, automation, and data science.

You: thanks
Bot: You're welcome!

You: bye
Bot: Goodbye! Have a great day.
```

```text
User: hello
```
